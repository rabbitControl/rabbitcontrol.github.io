RabbitControl:
A binary data-format definition to describe data values and user interface elements.
It is intended to expose parameters (values) from a host application to a client in a defined way.
UI clients can update values at the host application.

for more information please have a look at the projects home on github: https://github.com/rabbitControl
there you can also find the protocol specification: https://github.com/rabbitControl/RCP
